# Fetch API - Exercises

## Exercise 1: Basic Fetch
Create a function that fetches from a public API (like JSONPlaceholder) and returns the data.

## Exercise 2: POST Request
Create a function that sends a POST request with JSON data.

## Exercise 3: Error Handling
Build a robust fetch function that:
1. Checks response.ok
2. Handles different status codes
3. Returns proper error messages

## Exercise 4: Loading State
Create a function with a loading state that shows/hides a loading indicator.

## Exercise 5: Timeout
Implement a fetch with timeout functionality using AbortController.

## Exercise 6: Retry on Failure
Create a fetch wrapper that retries 3 times on failure.

## Exercise 7: Pagination
Create a function that fetches paginated data from an API.

## Exercise 8: Build a User Card
Create a mini app that:
1. Fetches a random user from randomuser.me API
2. Displays user info as a card
3. Has a "New User" button
