# Events - Exercises

## Exercise 1: Click Counter
Create a page with a button that tracks how many times it's been clicked. Display the count.

## Exercise 2: Keyboard Events
Create an input field that:
1. Shows the current key pressed
2. Shows how many characters have been typed
3. Prevents typing numbers (0-9)

## Exercise 3: Mouse Tracker
Track and display mouse position on the page. Show X and Y coordinates.

## Exercise 4: Form Validation
Create a form with:
- Name input (required)
- Email input (required, must contain @)
- Submit button

Show validation messages in real-time.

## Exercise 5: Event Delegation
Create a list where clicking any item logs its text. Use event delegation on the parent.

## Exercise 6: Tab Switching
Create tabs:
1. Click tab to show corresponding content
2. Update active tab styling
3. Only one tab active at a time

## Exercise 7: Drag and Drop Basics
Create a draggable element that logs "Dragging" when dragged and "Dropped" when dropped.

## Exercise 8: Debounce
Implement a debounce function that limits how often a function can be called.
