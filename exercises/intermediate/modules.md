# Modules & Error Handling - Exercises

## Exercise 1: Module Pattern
Create a Calculator module with:
- add, subtract, multiply, divide methods
- Private state for the result
- Public methods to interact

## Exercise 2: Custom Errors
Create custom error classes:
- ValidationError (with field property)
- NetworkError (with status code)
- Use them in a form validation scenario

## Exercise 3: Error Boundary
Create a wrapper function that catches all errors and logs them with context.

## Exercise 4: try/catch
Create async functions that demonstrate:
1. Successful try block
2. Caught error
3. finally block execution

## Exercise 5: Error Recovery
Create a function that tries multiple approaches and falls back gracefully.

## Exercise 6: Promise Error Handling
Handle errors in:
1. Promise chain with .catch()
2. try/catch with async/await
3. Promise.allSettled for partial failures
