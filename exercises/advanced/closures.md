# Closures & Scope - Exercises

## Exercise 1: Basic Closure
Create a function that returns a "secret" message, but the message should only be accessible through the returned function.

## Exercise 2: Counter Factory
Create `createCounter()` that returns:
- increment()
- decrement()
- reset()
- getCount()
Each counter should have independent state.

## Exercise 3: Memoize
Implement a `memoize(fn)` function that caches results.

## Exercise 4: Private Variables
Create a BankAccount class (using closures) with:
- Private balance
- deposit(amount)
- withdraw(amount)
- getBalance()

## Exercise 5: Loop Closure
Using setTimeout in a loop, print 0, 1, 2, 3, 4 after 1 second each. (Don't print 5 five times!)

## Exercise 6: Scope Chain
Given nested functions, trace through the scope chain to determine what variables are accessible.
