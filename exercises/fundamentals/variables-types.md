# Variables and Types - Exercises

## Exercise 1: Variable Declaration
Create variables using `let` and `const`:
1. Declare a constant for your name
2. Declare a variable for your age using let
3. Declare a variable for your favorite color using const
4. Try to reassign each variable and note what happens

## Exercise 2: Data Types
Identify the data type for each value:
```javascript
const values = [
  "Hello",
  42,
  3.14,
  true,
  null,
  undefined,
  { name: "Alice" },
  [1, 2, 3]
];
```
Create a function that returns the type of each value.

## Exercise 3: Type Conversion
Write functions to:
1. Convert a number to a string
2. Convert a string to a number
3. Convert any value to a boolean
4. Parse a float from "3.14" and an integer from "42"

## Exercise 4: Template Literals
Create a biography template using template literals:
```javascript
const name = "Your Name";
const age = 25;
const job = "Developer";
const city = "New York";
```
Output: "Hi, I'm [name], a [age] year old [job] living in [city]!"

## Exercise 5: Calculator State
Create a simple calculator state object with:
- currentValue
- previousValue
- operator
- Should use const for the object and let for mutable properties
