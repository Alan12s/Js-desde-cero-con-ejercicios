# Loops - Exercises

## Exercise 1: Count to 10
Use a for loop to print numbers 1 to 10.

## Exercise 2: Sum Array
Write a function `sumArray(arr)` that returns the sum of all numbers in an array.
Example: `sumArray([1, 2, 3, 4])` → 10

## Exercise 3: Find Maximum
Write a function `findMax(arr)` that returns the largest number in an array.
Example: `findMax([3, 7, 2, 9, 1])` → 9

## Exercise 4: Count Vowels
Write a function `countVowels(str)` that counts vowels in a string.
Example: `countVowels("Hello World")` → 3

## Exercise 5: Reverse Array
Write three versions of a function to reverse an array:
1. Using a for loop
2. Using while loop
3. Using for...of

## Exercise 6: Fibonacci
Write a function `fibonacci(n)` that returns the first n Fibonacci numbers.
Example: `fibonacci(7)` → [0, 1, 1, 2, 3, 5, 8]

## Exercise 7: Break and Continue
Write a function `findFirstEven(arr)` that finds the first even number but stops if none found.

## Exercise 8: Nested Loop - Multiplication Table
Create a 5x5 multiplication table:
```
1  2  3  4  5
2  4  6  8  10
3  6  9  12 15
4  8  12 16 20
5  10 15 20 25
```
