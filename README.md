# JavaScript Learning Repository

A comprehensive, structured path to master JavaScript - from fundamentals to pre-framework mastery.

## Overview

This repository is designed to take you from a JavaScript beginner to an intermediate/advanced level, preparing you to confidently learn frameworks like React, Vue, or Angular.

## Repository Structure

```
javascript-mastery/
├── docs/                    # Theoretical explanations
│   ├── fundamentals/       # Beginner topics
│   ├── intermediate/       # Intermediate topics
│   └── advanced/           # Pre-framework topics
├── examples/               # Code examples for each topic
├── exercises/             # Practice problems
├── solutions/             # Solutions to exercises
├── projects/              # Real-world projects
│   ├── beginner/          # Projects for beginners
│   ├── intermediate/       # Projects for intermediates
│   └── advanced/          # Pre-framework projects
├── projects_ideas.md      # Additional project ideas
└── README.md
```

## Learning Path

### Phase 1: Fundamentals (Beginner)

Master the building blocks of JavaScript:

| Topic | Description | Est. Time |
|-------|-------------|-----------|
| [Variables & Types](./docs/fundamentals/variables-types.md) | `let`, `const`, data types | 2 hours |
| [Operators](./docs/fundamentals/operators.md) | Arithmetic, comparison, logical | 1 hour |
| [Conditionals](./docs/fundamentals/conditionals.md) | `if/else`, `switch`, ternary | 2 hours |
| [Loops](./docs/fundamentals/loops.md) | `for`, `while`, `do...while` | 2 hours |
| [Functions](./docs/fundamentals/functions.md) | Declaration, expression, arrow functions | 3 hours |
| [Arrays & Objects](./docs/fundamentals/arrays-objects.md) | Collections and data structures | 4 hours |

### Phase 2: Intermediate JavaScript

Add interactivity and real-world functionality:

| Topic | Description | Est. Time |
|-------|-------------|-----------|
| [DOM Manipulation](./docs/intermediate/dom-manipulation.md) | Select, create, modify elements | 4 hours |
| [Events](./docs/intermediate/events.md) | Event listeners, bubbling, delegation | 3 hours |
| [Modules](./docs/intermediate/modules.md) | ES6 modules, import/export | 2 hours |
| [Async/Await & Promises](./docs/intermediate/async-await.md) | Asynchronous JavaScript | 4 hours |
| [Fetch API](./docs/intermediate/fetch-api.md) | HTTP requests, APIs | 3 hours |
| [Error Handling](./docs/intermediate/error-handling.md) | Try/catch, custom errors | 2 hours |

### Phase 3: Advanced Preparation (Pre-Framework)

Essential concepts that frameworks are built upon:

| Topic | Description | Est. Time |
|-------|-------------|-----------|
| [Closures](./docs/advanced/closures.md) | Lexical scoping, practical uses | 3 hours |
| [Scope](./docs/advanced/scope.md) | Global, local, block scope | 2 hours |
| [ES6+ Features](./docs/advanced/es6-features.md) | Destructuring, spread, classes | 4 hours |
| [Array Methods](./docs/advanced/array-methods.md) | map, filter, reduce, more | 4 hours |
| [Clean Code](./docs/advanced/clean-code.md) | Best practices, patterns | 3 hours |

## How to Use This Repository

### For Beginners
1. Start with the [Variables & Types](./docs/fundamentals/variables-types.md) documentation
2. Study the [examples](./examples/fundamentals/) for each topic
3. Complete the [exercises](./exercises/fundamentals/) to practice
4. Check [solutions](./solutions/fundamentals/) only after attempting
5. Build the [beginner projects](./projects/beginner/)

### For Intermediate Learners
1. Review the [fundamentals](./docs/fundamentals/) if needed
2. Progress through [intermediate topics](./docs/intermediate/)
3. Build [intermediate projects](./projects/intermediate/)

### For Pre-Framework Preparation
1. Master [advanced concepts](./docs/advanced/)
2. Complete [advanced projects](./projects/advanced/)
3. Explore [additional project ideas](./projects_ideas.md)
