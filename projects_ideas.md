# Additional Project Ideas

A comprehensive list of project ideas organized by difficulty level to continue your JavaScript learning journey.

---

## Beginner Projects

### 1. Color Picker
**Description**: An interactive color picker with RGB/HSL sliders and hex input.
**Features**:
- RGB sliders
- Color preview
- Hex code display
- Copy to clipboard
- Color history
**Skills**: DOM manipulation, events, CSS styling

### 2. Quiz App
**Description**: A multiple-choice quiz application with scoring.
**Features**:
- Question bank
- Multiple choice answers
- Score tracking
- Timer per question
- Results summary
**Skills**: Arrays, objects, conditionals, state management

### 3. Random Quote Generator
**Description**: Display random inspirational quotes.
**Features**:
- Fetch quotes from API
- Display quote and author
- Copy quote button
- Share on social (concept)
- New quote button
**Skills**: Fetch API, async/await, DOM manipulation

### 4. Password Generator
**Description**: Generate secure random passwords.
**Features**:
- Length slider
- Character type toggles (uppercase, lowercase, numbers, symbols)
- Copy to clipboard
- Password strength indicator
- Generate multiple passwords
**Skills**: String manipulation, arrays, events

### 5. Stopwatch
**Description**: A functional stopwatch with lap times.
**Features**:
- Start/Stop/Reset
- Lap functionality
- Lap times list
- Format time (MM:SS:MS)
**Skills**: Timing functions, DOM manipulation, arrays

### 6. BMI Calculator
**Description**: Calculate and categorize BMI.
**Features**:
- Height and weight inputs
- Metric/Imperial toggle
- BMI result and category
- Visual indicator
- Health recommendations
**Skills**: Functions, conditionals, math operations

### 7. Image Gallery
**Description**: A responsive image gallery with lightbox.
**Features**:
- Grid layout
- Click to enlarge
- Navigation arrows
- Close on click outside
- Keyboard navigation
**Skills**: DOM manipulation, events, CSS grid

### 8. Form Validator
**Description**: A form with real-time validation.
**Features**:
- Name, email, password, confirm password fields
- Real-time validation
- Error messages
- Success indicators
- Submit only when valid
**Skills**: Events, regex, conditionals, DOM manipulation

### 9. Memory Game
**Description**: A classic memory card matching game.
**Features**:
- Grid of face-down cards
- Click to reveal
- Match pairs
- Move counter
- Win detection
**Skills**: Arrays, DOM manipulation, game logic

### 10. Simple Drawing App
**Description**: A canvas-based drawing application.
**Features**:
- Draw with mouse
- Color picker
- Brush size
- Clear canvas
- Save as image
**Skills**: Canvas API, events, mouse tracking
